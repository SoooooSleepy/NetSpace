# Приложение для мониторинга трафика и управления сетевыми параметрами на WPF(C#) 

>[!WARNING]
>**WPF работает исключительно на Windows!** WPF жестко привязан к системным библиотекам Windows и используют исключительно DirectX, который не поддерживается в macOS и Linux. 

<img width="1792" height="817" alt="2026-07-09_02-33-42" src="https://github.com/user-attachments/assets/7fac722c-ffe1-4ccd-96e7-b2f8243948a8" />

**Функционал приложения:**
1) Получение данных об отправленных и полученных данных(также в виде графика), статусе подключения и активном сетевом интерфейсе
2) Отображение скорости сети
3) Управление сетевыми параметрами:
   a) DNS 
   b) Отображение информации о сетевых интерфейсах(вирт. и физ.), возможность изменять их статус
   c) IP-назначение(ipv4) 
   d) Дополнительные функции(очистка DNS-кеша и БД, исправление проблем с сетью)

>[!NOTE]
>**Изначально приложение запускается в фоне!** Основная часть данных для мониторинга расположены в контекстном окне.
